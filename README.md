# Resume Website (HTML Only)

This is a single-page resume website built using only HTML.

## Features
- Header with name
- About section
- Skills list
- Experience table
- Projects section
- Education table
- Contact details

## How to Run Locally
1. Download the repository
2. Open `index.html` in any browser
